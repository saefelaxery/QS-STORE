# LaLa Store - موقع بيع الملابس

## موقع مستقل تماماً - لا علاقة له بمشروع security-platform

### الموقع الجديد:
```
C:\Users\LaLa-Store\LaLaStore
```

### كيفية التشغيل:

1. افتح Terminal
2. اذهب إلى المجلد:
```bash
cd C:\Users\LaLa-Store\LaLaStore
```

3. شغل المشروع:
```bash
dotnet run
```

4. افتح المتصفح على:
   - **HTTP**: http://localhost:5051
   - **HTTPS**: https://localhost:7292

- ✅ المشروع الآن في مكان منفصل تماماً
- ✅ لا علاقة له بمشروع security-platform
- ✅ يمكنك حذف المجلد القديم من security-platform إذا أردت
